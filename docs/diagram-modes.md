# Diagram Modes — 다이어그램 모드 해석 가이드

## 개요

다이어그램은 선택한 건축물-건축가-도시 간의 관계를 시각화한다.
5개 모드는 **분석(Analysis) → 표현(Graphics)** 의 2단계 경험을 제공한다.

---

## Analysis 그룹 — 같은 데이터를 다른 렌즈로 분석

### 1. Force — "관계의 구조"

선택한 건축물-건축가-도시가 **서로 얼마나 연결되어 있는지**를 보여준다. 같은 건축가의 작품들은 자연스럽게 뭉치고, 관계가 약한 노드는 바깥으로 밀려난다.

**볼 것**: 클러스터(뭉침)의 형태. 한 건축가 작품만 모았으면 하나의 덩어리, 다양한 건축가를 선택했으면 여러 클러스터가 생긴다. 연결선의 밀도가 곧 "큐레이션의 응집도".

**시각 장치**: 빈 캔버스 + 곡선 링크 — 물리적 장력을 시각화하는 유기적 네트워크

### 2. Timeline — "시간 축의 분포"

X축이 건축물의 **완공 연도**가 된다. 선택한 건축물들이 시간적으로 어디에 분포하는지 한눈에 보인다.

**볼 것**: 특정 시대에 집중되어 있는지, 넓게 퍼져 있는지. "60년대 브루탈리즘 컬렉션"이면 왼쪽에 몰리고, "1960~2020 통시적 셀렉션"이면 고르게 분포한다. 시간 흐름에 따른 건축 스타일 변화를 읽을 수 있다.

**시각 장치**: 연도별 세로 룰드라인 + 하단 연도 축 라벨 + "YEAR →" 방향 표시

### 3. Geography — "공간적 배치"

X축이 경도, Y축이 위도 — 선택한 건축물들의 **실제 지리적 위치 관계**를 추상화한다.

**볼 것**: 유럽에 몰려있는지, 대륙을 넘나드는지. 같은 건축가의 작품이 어떤 도시에 분포하는지. 지도에서 보는 것보다 **관계선**이 추가되어 "이 건축가가 이 도시들을 연결한다"는 맥락이 드러난다.

**시각 장치**: 좌표 격자 그리드 + N/S/W/E 축 라벨 + 컴파스 마크(N↑)

---

## Graphics 그룹 — 분석 결과를 결과물로 전환

### 4. Blueprint — "건축 도면"

건축가를 왼쪽 열, 도시를 오른쪽 열에 놓고, 건축물들을 가운데 그리드에 정렬한다. 건축 도면의 **제도판** 같은 느낌.

**볼 것**: 결과물 자체가 목적. Force/Timeline으로 분석한 후, 그 셀렉션을 **깔끔하게 정리된 도면**으로 변환한다. PNG로 내보내면 A4 프린트용 레퍼런스 시트가 된다.

**시각 장치**: 밀도 높은 그리드(20px) + 이중선 프레임 + 우하단 타이틀 블록 ("ARCHI CURATION" + 날짜) + 직각 꺾은선 링크

### 5. Poster — "그래픽 인쇄물"

상단에 여백(제목 영역), 건축물을 카드형으로 배치, 하단에 건축가/도시를 나열한다. **인쇄 가능한 포스터 비율**.

**볼 것**: 역시 결과물 자체가 목적. "내가 큐레이션한 건축물 셀렉션"을 **저장하고 싶은 그래픽**으로 만든다. 전시 포스터, SNS 공유 이미지 감각.

**시각 장치**: 빈 배경 + 마진 프레임 + 상단 "CURATED SELECTION" 대형 타이포 + 링크 극히 희미하게

---

## 왜 각각 달라야 하는가

Analysis 3개는 **같은 데이터를 다른 렌즈로 분석**한다 — 관계/시간/공간. 이 세 축은 건축을 이해하는 근본적인 차원이다.

| 축 | 질문 | 모드 |
|---|---|---|
| 관계 | "이 건축물들은 어떻게 연결되는가?" | Force |
| 시간 | "이 건축물들은 언제 지어졌는가?" | Timeline |
| 공간 | "이 건축물들은 어디에 있는가?" | Geography |

Graphics 2개는 **분석 결과를 결과물로 전환**한다. 사용자가 "이 셀렉션이 재밌다"고 느끼면 → Blueprint/Poster로 **그래픽 산출물**을 만든다. 이것이 PRD에서 말하는 "선택을 그래픽 결과물로 바꾸는 다이어그램형 인터랙션"의 핵심이다.

| 목적 | 감각 | 모드 |
|---|---|---|
| 도면으로 정리 | 건축 제도판, 레퍼런스 시트 | Blueprint |
| 포스터로 표현 | 전시 포스터, SNS 공유 | Poster |

즉 **분석(왜 이 건축물들인가) → 표현(이 큐레이션을 어떻게 보여줄 것인가)** 의 2단계 경험을 하나의 다이어그램 안에서 모드 전환으로 제공하는 구조다.

|---|---|---|---|
| Building | ● 원형 (r=16) | 건축가 색상 | 선택된 건축물 |
